# ToDoListApp

A simple and elegant To-Do List web application built using HTML, CSS, and JavaScript. This app helps users manage daily tasks efficiently with a clean interface and intuitive controls.

# Features

- **Add Tasks**
- **Mark Completed**
- **Delete Tasks**
- **UI Updates**
- **Responsive Design**
